MCP Protocol

MCP 소개

Model Context Protocol (MCP)는 Anthropic에서 2024년 말에 도입한 새로운 개방형 표준으로, AI 애플리케이션이 다양한 외부 도구, 데이터 소스, 시스템과 원활히 상호 작용할 수 있도록 설계되었습니다. 이는 기존의 파편화된 통합 방식을 대체하고, 도구와 데이터 소스에 대한 보편적인 프레임워크를 제공하여 AI 시스템이 다양한 컨텍스트에 쉽게 접근하도록 지원합니다. MCP는 AI 통합을 위한 USB-C와 같은 역할을 하며, 기존의 복잡한 M×N 문제를 단순한 M+N 문제로 변환합니다.

출처

MCP의 구조와 작동 원리

MCP는 Host-Client-Server의 3층 구조로 설계되었으며 JSON-RPC로 통신합니다.
